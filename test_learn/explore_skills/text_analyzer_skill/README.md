# Text Analyzer Skill

## 快速开始

1. 确保已安装Python 3.10+
2. 如需中文分析，安装jieba: `pip install jieba`

## 测试脚本

```bash
python scripts/analyzer.py
```

## 目录结构

```
text_analyzer_skill/
├── SKILL.md              # 技能说明文档
├── README.md             # 本文件
├── scripts/              # 执行脚本
│   └── analyzer.py       # 核心分析器
└── references/           # 参考资源
    └── stopwords.txt     # 停用词列表
```
